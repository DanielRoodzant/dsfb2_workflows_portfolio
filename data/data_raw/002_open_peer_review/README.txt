These files belong to the chapter Open Peer Review of the portfolio. 

bats_and_covid.R is a dataset belonging to the following project found in the ORF database: https://osf.io/wxh6a/
It is used in the chapter to reproduce some of the code found within the file. The functionality of the .R file is also described in the chapter.

GDELTBatsUS1620.csv is the file that is used by the reproduced code to generate a graph and was found on the same URL as the file above. The data is about US Tv news on bats between 2016 and 2020.

reproducibility_criteria.xlsx is an Excel file containing the table shown at the start of the chapter that displays the reproducibility criteria.
